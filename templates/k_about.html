KINTSUGI

This project is built with an aim of "MENS SANA IN CORPORE SANO" which means a healthy mind in a healthy body.

Our Work

01. Data Collection

We are using data of Twitter which is open-source. You can also view sample dataset in Data Preview section.

02. Train a Model

After completion of data pre-processing, final model is stored based on some evaluation criteria.

03. Prediction

Based on stored model, prediction will be made on user input(unseen data) which you can see in Prediction section of this portal.
